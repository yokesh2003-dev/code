🧠 AI TASK ENGINE

Convert input (text OR transcript) into structured tasks

Each task must include:

* id
* title
* description
* owner
* assignedBy
* deadline
* priority (Urgent / Important / Later / Optional)
* category (Work / Follow-up / Review / Personal)
* effort (Low / Medium / High)
* status (Pending / Completed / Expired)
* workspaceId
* comments
* ccList

Rules:

* Detect names → assign owner
* Detect deadlines → assign urgency
* "today" → Urgent
* action words → Important
* vague → Later
* optional tone → Optional
